# PWA Icons

Remplacez ces fichiers par vos icônes réelles. Tailles attendues :

- `icon-192.png` — 192×192 (PNG, sans transparence, fond `#F4F3F0` ou `#0A0908`)
- `icon-512.png` — 512×512
- `maskable-192.png` — 192×192 (zone de sécurité 80% centrée — voir [maskable.app](https://maskable.app))
- `maskable-512.png` — 512×512
- `apple-touch-icon.png` — 180×180 (PNG, fond opaque, coins arrondis auto par iOS)
